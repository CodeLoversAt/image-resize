# image-resize
little package to do image resizing and optimization with gulp

## Installation

```shell
git clone https://github.com/CodeLoversAt/image-resize.git
cd image-resize
npm install
```

## Usage

All options show the default value and are optional.

```shell
gulp --width=1200 \
    --height=0 \
    --crop=true \
    --upscale=false \
    --optimizationLevel=5 \
    --progressive=true \
    --interlaced=true
```
